# Subwasm

## Installation

### Using Cargo

```sh
cargo install --locked --git https://github.com/chevdor/subwasm --tag v0.16.1
```

### Using Homebrew for macOS

```sh
brew tap chevdor/subwasm https://github.com/chevdor/subwasm
brew install subwasm
```
